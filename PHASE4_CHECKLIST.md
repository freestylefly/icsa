## ✅ 1. 人工协作模块（权重：30%）

### 1.1 坐席工作台
- [ ] 坐席可以正常登录工作台
- [ ] 会话列表显示正确（包含未读消息数）
- [ ] 聊天窗口消息收发正常
- [ ] 消息支持富文本和表情
- [ ] 会话切换流畅无卡顿

**测试方法**: 使用 3 个以上坐席账号同时登录，进行会话测试

### 1.2 坐席状态管理
- [ ] 坐席状态切换正常（在线/忙碌/离线/休息）
- [ ] 状态变更后实时同步到其他坐席
- [ ] 离线坐席不接收新会话
- [ ] 忙碌坐席不接收新会话（除非强制分配）

**测试方法**: 
1. 坐席 A 切换到忙碌状态
2. 坐席 B 创建新会话
3. 验证会话只分配给在线坐席

### 1.3 会话分配策略
- [ ] 轮询策略：会话均匀分配给坐席

- [ ] 技能匹配：优先分配给有对应技能的坐席

**测试方法**: 创建 10 个会话，验证分配是否合理

### 1.4 实时消息推送
- [ ] WebSocket 连接稳定
- [ ] 消息推送延迟 < 100ms
- [ ] 断线自动重连
- [ ] 重连后消息不丢失

**测试方法**: 使用网络工具模拟断线，验证重连机制

### 1.5 对话摘要生成

- [ ] 关键点提取合理（3-5 个）

- [ ] 分类和建议合理

**测试方法**: 生成 10 个对话摘要，人工评估准确性
