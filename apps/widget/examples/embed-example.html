🚀 快速开始

  1. 引入 Widget 脚本

    在 HTML 的 <head> 或 <body> 结束标签前添加：

    <!-- 生产环境 -->
<script src="https://your-domain.com/widget.js"></script>

<!-- 开发环境 -->
<script src="http://localhost:3002/widget.js"></script>

  2. 初始化 Widget

    在页面加载完成后调用初始化函数：

    <script>
  // 等待 DOM 加载完成
  document.addEventListener('DOMContentLoaded', function() {
    // 初始化 Widget
    IntelligentCustomerServiceWidget.init({
      apiBaseUrl: 'https://your-api.com',  // 后端 API 地址
      tenantId: 'your-tenant-id',            // 租户 ID（必填）
      agentId: 'your-agent-id',              // 坐席 ID（可选）
      theme: {
        primaryColor: '#1890ff',             // 主题色
        position: 'right',                    // 位置：'left' 或 'right'
      },
    });
  });
</script>

  3. 完成！

    Widget 会自动出现在页面右下角，用户可以点击开始对话。

📚 API 文档

初始化配置

参数 类型 必填 说明
apiBaseUrl string 是 后端 API 服务地址
tenantId string 是 租户唯一标识
agentId string 否 坐席 ID，用于标识客服
theme.primaryColor string 否 主题颜色，默认 #1890ff
theme.position string 否 Widget 位置，left 或 right，默认 right

控制方法

// 打开聊天窗口
IntelligentCustomerServiceWidget.open();

// 关闭聊天窗口
IntelligentCustomerServiceWidget.close();

// 销毁 Widget（移除 DOM）
IntelligentCustomerServiceWidget.destroy();

🎨 自定义样式

虽然 Widget 使用 Shadow DOM 隔离样式，但您仍然可以通过 CSS 变量自定义部分外观：

<style>
  /* 自定义 Widget 主题色 */
  :root {
    --ic-primary-color: #1890ff;
    --ic-success-color: #52c41a;
    --ic-error-color: #ff4d4f;
    --ic-warning-color: #faad14;
  }
</style>

⚠️ 注意事项

💡 提示： Widget 会自动保存会话 ID 到 localStorage，用户刷新页面后会话会自动恢复。
🔒 安全： 请确保使用 HTTPS 协议部署 Widget，以保护用户数据安全。
✅ 性能： Widget 使用懒加载和按需连接策略，不会影响页面加载性能。

🔧 故障排查

Widget 不显示

  • 检查是否正确引入了 widget.js 脚本
  • 检查浏览器控制台是否有错误信息
  • 确认 tenantId 配置正确

无法连接服务器

  • 检查 apiBaseUrl 是否正确
  • 确认后端服务已启动
  • 检查网络连接和 CORS 配置

样式问题

  • Widget 使用 Shadow DOM，应该完全隔离
  • 如有样式冲突，检查是否有全局样式污染
